  <i>CLI tool to provide custom configurations on the go</i>
</p>

## Usage

Show commands available:

```bash
$ npx punk --h
```

Use command:

```bash
$ npx punk <command>
```

You can pass arguments to commands too:

```bash
$ npx punk prettier -h
```
